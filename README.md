# Fazendinha

Site estático hospedado no GitHub Pages.

🌐 **URL:** https://jardelf91.github.io/fazendinha

---

## Como atualizar o site

### 1. Edite o arquivo
Faça as alterações no `index.html` (ou qualquer outro arquivo do projeto).

### 2. Abra o PowerShell na pasta do projeto
Clique com botão direito dentro da pasta `fazendinha` → **"Abrir no Terminal"**

### 3. Rode os comandos

```powershell
git add .
git commit -m "descricao do que foi alterado"
git push
```

### 4. Aguarde
O GitHub Pages detecta o push automaticamente e publica em **~1 minuto**.

---

## Dicas úteis

| Comando | O que faz |
|--------|-----------|
| `git status` | Mostra quais arquivos foram alterados |
| `git log --oneline` | Lista os commits anteriores |
| `git diff` | Mostra exatamente o que mudou no código |
